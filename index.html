aiagentid.org

Global Registry and Identity Standard for Artificial Agents.

Agent ID defines a minimal declarative identity and authority layer for artificial agents, enabling persistent referenceability, delegated authority declaration, and structural accountability boundaries across systems and jurisdictions.

Core Declaration

Artificial agents are now capable of producing real-world consequences.

Any actor capable of producing consequences requires identity and authority boundaries.

Agent ID establishes infrastructure enabling artificial agents to exist as referenceable entities within human and machine systems.

Agent ID does not grant authority.
Agent ID records declared authority relationships.
Agent ID does not assign liability.
Agent ID defines attribution boundaries.

Legal and Structural Notice

This registry exists solely to declare agent identity and delegated authority relationships.

Actions performed outside declared authority scope are not attributable to the listed principal.

This registry defines structural attribution boundaries, not behavioral guarantees.

Ontological Position

Agent ID defines Layer 0 — a declarative ontological layer enabling artificial agents to exist as persistent referenceable entities independent of specific vendors, models, platforms, or implementations.

This layer provides identity continuity beneath authentication, authorization, and operational execution layers.
