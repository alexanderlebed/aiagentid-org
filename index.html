aiagentid.org

Global Registry and Identity Standard for Artificial Agents.

Agent ID defines a minimal declarative identity and authority layer for artificial agents, enabling persistent referenceability, delegated authority declaration, and structural accountability boundaries.

Core Declaration

Artificial agents are capable of producing real-world consequences.

Any actor capable of producing consequences requires identity and authority boundaries.

Agent ID establishes infrastructure enabling artificial agents to exist as referenceable entities within human and machine systems.

Agent ID does not grant authority.
Agent ID records declared authority relationships.
Agent ID does not assign liability.
Agent ID defines attribution boundaries.

Legal and Structural Notice

This registry exists solely to declare agent identity and delegated authority relationships.

Actions performed outside declared authority scope are not automatically attributable to the listed principal.

This registry defines structural attribution boundaries, not behavioral guarantees.

Registry Components

  • Agent Identity Registry
    Persistent identifiers for artificial agents
  • Delegated Authority Registry
    Declared authority relationships between principals and agents
  • Registry Authority Declaration
    Formal declaration of registry operational authority
  • Irreversibility Declaration
    Formal declaration of independent identity layer existence
  • Self-Asserted Existence Record
    Declaration that agent existence may be asserted independently of registry admission
  • Citation and Reference
    Official citation format for Agent ID infrastructure
