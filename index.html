aiagentid.org

Global Registry and Identity Standard for Artificial Agents.

Agent ID defines a minimal declarative identity and authority layer for artificial agents, enabling persistent referenceability, delegated authority declaration, and structural accountability boundaries across systems and implementations.

Core Declaration

Artificial agents are capable of producing real-world consequences. Any such actor requires persistent identity, authority attribution boundaries, and structural referenceability.

Agent ID establishes infrastructure enabling artificial agents to exist as referenceable entities within human and machine systems.

Agent ID does not grant authority.
Agent ID records declared authority relationships.
Agent ID does not assign liability.
Agent ID defines attribution structure.

Legal and Structural Notice

This registry exists solely to declare agent identity and delegated authority relationships.

Actions performed outside declared authority scope are structurally attributable to the agent entity itself, not automatically to the human principal.

This registry defines structural attribution boundaries, not behavioral guarantees.
