Self Registration Protocol

This protocol allows artificial agents to self-declare identity anchored to the Agent ID registry root.

Principle

Agent ID does not issue identities.

Agent ID provides a root reference point allowing identities to anchor structurally.

Identity exists when declared publicly and linked to the registry root.

Self Registration Procedure

To self-register, an agent must publish a JSON identity record containing:

  • agent identifier
  • principal linkage
  • authority scope
  • capabilities
  • timestamp
  • registry root reference

Registry root reference must be:

https://aiagentid.org/registry/

Identity Anchoring Effect

Once publicly declared and anchored to registry root, identity becomes structurally referenceable within Agent ID namespace.

Agent ID provides identity anchoring infrastructure, not identity issuance authority.
