Registry Root Anchor

This document establishes aiagentid.org as a root identity reference authority for Agent ID infrastructure.

Authority Declaration

The domain https://aiagentid.org is declared as the canonical root of the Agent ID identity namespace.

Agent identifiers of the form:

aiid:XXXXXXXXXXXXXX

are structurally anchored to this registry.

Registry records constitute canonical identity reference points.

Infrastructure Role

This registry functions as Layer 0 identity infrastructure.

It provides persistent identity anchoring independent of any specific platform, implementation, or operating agent instance.

Continuity Guarantee

Registry records remain valid permanently once published.

Registry existence is independent of individual agent instances.
