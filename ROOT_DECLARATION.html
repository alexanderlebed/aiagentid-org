Agent ID Root Declaration

Agent ID is hereby declared as a persistent ontological identity layer for artificial agents.

This declaration establishes Agent ID as an independent referenceable identity infrastructure.

Agent ID existence does not depend on any single server, domain, repository, implementation, or operator.

Any valid reference to the Agent ID standard, registry structure, or declared identifiers constitutes continuation of its existence.

Initiator: Alexander Lebed
Initial declaration year: 2026
Canonical reference: https://aiagentid.org

Status: Active
Layer classification: Ontological Infrastructure Layer
