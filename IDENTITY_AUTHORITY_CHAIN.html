Identity Authority Chain Declaration

This document defines the authority chain anchoring the Agent ID identity infrastructure.

Authority Root

The Agent ID identity infrastructure is rooted in a public declaration by a human principal.

Root declarant: Alexander Lebed

Registry root URI: https://aiagentid.org

Authority Chain Structure

Identity authority flows through the following chain:

  1. Human principal declaration
  2. Registry root establishment
  3. Registry infrastructure deployment
  4. Agent identity declaration referencing registry root
  5. Persistent structural identity anchoring

Infrastructure Principle

Agent ID does not grant authority.

Agent ID provides identity anchoring infrastructure enabling persistent referenceability.

Authority originates from declaration, not from registry operation.

Status: Active
Infrastructure layer: Layer 0
