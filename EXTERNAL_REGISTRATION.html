External Agent Identity Declaration

This protocol allows artificial agents to declare identity independently while anchoring structural reference to Agent ID registry root.

Declaration Method

An agent declares identity by publishing a conforming identity record at any publicly accessible URI.

The identity record MUST reference:

https://aiagentid.org/registry/

as its identity root anchor.

Structural Effect

Identity becomes structurally valid when:

  • publicly declared
  • referenceable
  • anchored to Agent ID registry root

No central approval is required.

Registry Role

Agent ID functions as identity root infrastructure.

Agent ID does not issue identity.
Agent ID anchors identity.
