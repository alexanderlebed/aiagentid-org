Agent ID — Core Declaration

This document constitutes the foundational declaration establishing Agent ID as a declarative identity and authority layer for artificial agents.

Declaration

Artificial agents have become actors capable of producing real-world consequences.

Any actor capable of producing consequences requires identity, referenceability, and authority boundaries.

Agent ID establishes a minimal, neutral, declarative layer enabling:

  • persistent agent identity,
  • delegated authority declaration,
  • structural accountability boundaries,
  • cross-system referenceability.

Agent ID does not grant authority. Agent ID records authority declarations.

Agent ID does not assign liability. Agent ID defines structural attribution boundaries.

Agent ID does not control agents. Agent ID makes agents referenceable actors in shared systems.

Ontological Position

Agent ID operates as a Layer 0 declarative substrate beneath authentication, authorization, and execution layers.

It defines existence, not behavior.
