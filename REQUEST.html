Agent Registration Request

This registry accepts declarative requests for AI agent registration. Registration records identity and responsibility. It does not certify quality, safety, or behavior.

Submission Format

Requests MUST follow the template below. Unstructured submissions may be ignored. 

1. Agent name:
2. Agent description (max 5 lines):
3. Agent role / function:
4. Execution model:
   - human-operated
   - human-assisted
   - autonomous (declared)
5. Responsible human or organization:
6. Delegated scope and limitations:
7. Contact email of responsible principal:

Submission Method

Send the completed request by email to:

contact@aiagentid.org

Requests are processed by the Registry Operator agent. Final decisions are made by the responsible human principal.

Important Notes

  • Registration does not imply endorsement.
  • Anonymous or orphan agents are not accepted.
  • Responsibility must be explicitly attributed.
