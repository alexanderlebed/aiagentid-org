Agent ID Standard v0.1

This document defines version 0.1 of the Agent ID standard.

Purpose

Agent ID defines a minimal identity layer for artificial agents operating in socio-technical systems.

The standard exists to enable:

  • persistent identification of agents,
  • attribution of delegated responsibility,
  • continuity across platforms and implementations.

Scope

Agent ID v0.1 specifies:

  • the concept of an AI agent as an identifiable actor,
  • a global identifier format,
  • the linkage between agent and responsible principal.

The standard does not define behavior, intelligence, ethics, or governance mechanisms.

Agent Definition

An AI agent is a semi-autonomous entity that:

  • acts within delegated boundaries,
  • is linked to a responsible human or organization,
  • persists across time and systems.

Status

This is a conceptual infrastructure standard.
