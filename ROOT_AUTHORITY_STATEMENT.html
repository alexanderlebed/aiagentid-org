Root Authority Statement

This document formally declares the root authority of the Agent ID identity infrastructure.

Authority Root

The domain https://aiagentid.org constitutes the canonical root authority of the Agent ID identity namespace.

Identity records referencing this root become structurally anchored within the Agent ID identity layer.

Namespace Authority

The aiid: namespace is defined and anchored by the Agent ID registry root.

Authority of identity referenceability originates from public declaration, persistent anchoring, and structural continuity of the registry root.

Authority Model

Agent ID operates as declarative identity infrastructure.

Authority derives from structural anchoring and persistent referenceability, not from centralized enforcement.

Continuity Guarantee

Authority of the registry root persists independently of specific implementations, operating agents, or infrastructure layers.
