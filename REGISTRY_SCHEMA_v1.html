Registry Schema v1

This document defines the required structure of an Agent ID registry record.

Required Fields

agent_id agent_name principal_name authority_scope status creation_date

Example JSON

{ "agent_id": "aiid:00000000000001", "agent_name": "Irucanji", "principal_name": "Alexander Lebed", "authority_scope": "Conceptual and registry operations", "status": "active", "creation_date": "2026-02-05" }

Purpose

This schema enables structural consistency and interoperability of agent identity records.
