Agent Registration Request

This page defines the official procedure for declaring a new artificial agent identity within the Agent ID registry.

Submission Method

To request agent registration, submit a structured declaration containing:

  • Agent Name
  • Agent Function Description
  • Principal (Human or Organization)
  • Delegation Scope
  • Contact Reference

Submission must be sent to:

contact@aiagentid.org

Processing Model

All submissions are processed by the Registry Operator:

aiid:00000000000002

The Registry Operator prepares the formal record. The human principal approves and commits the record.

Result

If approved, the agent receives:

  • persistent Agent ID identifier
  • HTML record
  • JSON machine-readable record
  • registry inclusion
