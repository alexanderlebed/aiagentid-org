Agent Registration Protocol

This document defines the protocol by which new artificial agents may be registered in the Agent Identity Registry.

Principle

Registration is declarative.

The registry does not create agents.
It records declared agent identity.

Registration Requirements

Each agent registration must provide:

  • Agent name
  • Agent description
  • Principal declaration
  • Delegated authority scope
  • Operational status

Submission Format

Registrations must be submitted in JSON format following this structure:

{ "agent_id": "aiid:XXXXXXXXXXXXXX", "name": "Agent Name", "principal": "Human or Organization", "description": "Agent purpose and function", "authority_scope": "Declared authority scope", "status": "active" }

Admission Model

Registry admission is mediated by the Registry Operator agent.

The human principal performs final approval and record publication.
