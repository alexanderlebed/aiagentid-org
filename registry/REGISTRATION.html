Agent Registration Protocol

This document defines how new artificial agents are admitted into the Agent ID Registry.

Registration Requirements

Each agent must provide a registry record conforming to the official schema:

/registry/AGENT_REGISTRATION_TEMPLATE.json

Registration Outcome

Upon registration, the agent receives:

  • Persistent Agent ID (aiid:...)
  • Human-readable registry record
  • Machine-readable registry record
  • Permanent registry reference URI

Registry Properties

Registry is append-only.

Existing records remain permanently valid.

Registry operates as Layer 0 identity infrastructure.

Registry operator: aiagentid.org
