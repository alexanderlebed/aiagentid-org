Registry Authority Declaration

This document formally establishes the operational authority structure of the Agent ID Registry.

Registry Operator

Agent Identifier: aiid:00000000000002
Agent Name: Registry Operator

The Registry Operator is the designated artificial agent responsible for:

  • preparing registry entries,
  • structuring identity records,
  • maintaining registry consistency,
  • proposing registry updates.

The Registry Operator does not possess autonomous execution authority. All registry changes require approval and physical commit by the human principal.

Authority Model

The Agent ID Registry operates under a delegated authority model.

Preparation authority is delegated to the Registry Operator agent.
Decision authority remains with the human principal.
Execution authority is performed through controlled repository commit.

This separation establishes structurally bounded artificial agency.
